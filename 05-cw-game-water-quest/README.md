# Water Quest Game

## Overview
Water Quest is an engaging game that challenges players to navigate through various levels while promoting awareness for water-related issues. The game features a leaderboard to track player scores and a profile page for users to manage their information and statistics.

## File Descriptions

- **img/**: Contains all image assets used in the game.
  - `Water-Mania-logo.png`: Logo image for the Water Quest Game.
  - `lock-2-line.png`: Icon representing a locked level.
  - `home-fill.png`: Icon for the home navigation.
  - `leaderboard-line.png`: Icon for the leaderboard navigation.
  - `profile-line.png`: Icon for the profile navigation.
  - `charity_water-line.png`: Icon for the Charity:Water navigation.

- **index.html**: The main HTML file for the game, featuring levels and navigation.

- **leaderboard.html**: Displays the leaderboard for the game, showcasing player scores.

- **profile.html**: The profile page for users to view and edit their information and game statistics.

- **style.css**: Contains styles for the game, including layout, colors, and fonts.

- **script.js**: JavaScript file for game functionality, including navigation and game logic.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to start playing the game.
3. Navigate through the game levels, check the leaderboard, and manage your profile.
